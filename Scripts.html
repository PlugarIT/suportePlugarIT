Scripts

- Compacta as pastas Ctrl Geral DefRel Empresas XSD e Empresas

  • Este Script faz a compactação das pastas acima citadas para facilitar a cópia das mesmas

INSTRUÇÕES

  • Entre na pasta C:\CSS\Scripts e copie o script Copia e Compacta as Pastas Ctrl Geral Defrel Scan Empresas xsd.bat para a máquina do cliente
  • Execute como Administrador
  • Ele irá pedir para digitar a unidade ou letra do mapeamento onde o sistema Phoenix esta instalado, exe C: ou Z:, etc.
  • Em seguida irá pedir o nome do diretório ou pasta onde o sistema Phoenix esta Instalado, ex. Phoenix.
  • O Script irá criar uma pasta dentro da pasta Phoenix e dentro irá criar os seguintes arquivos:

    • - Ctrl-2408-0950.rar

    - Empresas-2408-0950.rar

    - Geral-2408-0949.rar

    - Scan-2408-0950.rar

    - XSD-2408-0954.rar

  • O arquivo tem o nome da pasta compactada e em seguida a data e hora da geração do arquivo.
  • Caso você execute novamente o script ele irá apagar os arquivos gerados e fazer o processo novamente.

- Registra a DLL Midas.

  • Este Script faz o registro do Midas.dll

INSTRUÇÕES

  • Algumas vezes pode acontecer do Windows perder ou mesmo não registrar uma determinada DLL por motivos diversos, para facilitar esse registro fizemos este script que vai inserir no regstro do Windows os dados necessário para isso.
  • Entre na pasta C:\CSS\Scripts e copie o script REGISTRO_MIDAS_X32.bat ou REGISTRO_MIDAS_X64.bat dependendo do S.O. do cliente
  • Entre na pasta Phoenix e copie o Midas.dll para a pasta C:\Windows\System32 caso o S.O. for 32bits ou C:\Windows\SysWoW64\ caso seja 64bits
  • Execute o script como Administrador

- Apaga os anos especifícos das empresas.

  • Este Script apaga os anos especifícos conforme adicionado no script

INSTRUÇÕES

IMPORTANTISSÍMO:

  • Antes de executar esse processo faça um backup da pasta Empresas completo.
  • Entre na pasta C:\CSS\Scripts e copie o script Apaga Subpastas por Periodos.bat
  • Execute como Administrador
  • Ele irá pedir para digitar a unidade ou letra do mapeamento onde o sistema Phoenix esta instalado, exe C: ou Z:
  • Em seguida irá pedir o nome do diretório ou pasta onde o sistema Phoenix esta Instalado, ex. Phoenix.
  • Veja o exemplo nas 3 linhas abaixo para apagar os anos de 2000, 2001 e 2002.

for /F %%G in ('dir / a: d / s / b 2000*') do rd /s /q "%%G"

for /F %%G in ('dir / a: d / s / b 2001*') do rd /s /q "%%G"

for /F %%G in ('dir / a: d / s / b 2002*') do rd /s /q "%%G"

- Apaga LCKS.

  • Este Script apaga todos os LCKS da pasta Phoenix

INSTRUÇÕES

  • Entre na pasta C:\CSS\Scripts e copie o script Apaga LCKS.bat para máquina do cliente
  • Para que possa apagar todos os arquivos LCKs é nessessário que ninguem esteja utilizando os sistemas.
  • Execute como Administrador
