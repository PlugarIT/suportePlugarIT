PROCEDIMENTOS A SEREM APLICADOS PARA USUÁRIOS DO AD COM BTRIEVE 6.15

Antes de aplicar as permissões a hierarquia em cima de cada arquivo ou pasta deve ser quebrada.

Dar permissão total para todos nas pastas do registro do windows:

Para sistemas 32 bits:

  • HKEY_LOCAL_MACHINE\Software\Btrieve Technologies

  • HKEY_LOCAL_MACHINE\Software\Contmatic

Para sistemas 64 bits:

  • HKEY_LOCAL_MACHINE\Software\Wow6432Node\Contmatic

  • HKEY_LOCAL_MACHINE\Software\Wow6432Node\Btrieve Technologies

Dar permissão total para TODOS nas pastas: C:\TEMP e C:\TMP

Criar o arquivo texto configG5.ini no caminho abaixo indicado e dar permissão total para TODOS:

Para sistemas 32bits:

  • C:\WIDNOWS\SYSTEM32\ConfigG5.ini

Para sistemas 64bits:

  • C:\WINDOWS\SYSWOW64\ConfigG5.ini

Dar permissão TOTAL para TODOS nos arquivos:

  • C:\WINDOWS\MKDEM.LOG

  • C:\WINDOWS\MDKDEWE.TRN

Para sistemas 32bits:

  • C:\WINDOWS\SYSTEM32\W32MKDE.EXE

  • C:\WIDNOWS\SYSTEM32\W32MKRC.DLL

  • C:\WINDOWS\SYSTEM32\WBTRV32.DLL

Para sistemas 64bits:

  • C:\WINDOWS\SYSWOW64\W32MKDE.EXE

  • C:\WINDOWS\SYSWOW64\W32MKRC.DLL

  • C:\WINDOWS\SYSWOW64\WBTRV32.DLL

A UAC deve estar totalmente baixa no usuário da máquina local

Adicionar o diretório mapeado do sistema bem como o motor do banco nas excessões do antivírus no servidor e nos computadores também

Ajustar o mapeamento nas propriedades dos atalhos do sistema de acordo com a letra meapeda pelo Windows
