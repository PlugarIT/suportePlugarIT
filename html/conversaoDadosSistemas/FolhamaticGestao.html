Para conversão de dados da Folhamatic Gestão é obrigatório que o SQL Server esteja instalado na máquina de conversão.

Para executar esta etapa supomos que o SQL Server já esteja instalado e configurado.

Começaremos criando o Data Base restaurando o backup recebido conforme a figura abaixo.

Para isso abra o SQL Server Management e clique com o botão direito do mouse em Banco de Dados do Sistema

Clique em Restaurar Arquivos e Grupos de Arquivos

Clique em Restaurar Arquivos e Grupos de Arquivos

Preencha os campos conforme figura abaixo e clique para selecionar a base a ser restaurada

Clique em Adicionar

Agora selecione o local dos dados a restaurar

Configure a tela como abaixo e clique em ok
