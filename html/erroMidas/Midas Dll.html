Midas Dll

Este erro acontece geralmente quendo após a instalação de algum sistema o Windows não consegue registrar a Dll Midas

01 - Erro Midas Dll Ao entrar no sistema

SOLUÇÃO:

  • Midas DLL Para solucionar copie o Midas.Dll da pasta Phoenix para a pasta C:\Windows\SysWOW64.

  • Caso os sistema operacional seja 64bits (Se o S.O. for 32 Bits copie o Midas para a pasta C:\Windows\System32.

  • Registre manualmente essa Dll, para isso abra a pasta C:\CSS\Scripts e Execute o arquivo REGISTRO MIDAS X64.bat ou REGISTRO MIDAS X32.bat (32Bits)como administrador na máquina que estiver ocorrendo o erro, no mesmo local tem o arquivo PassoAPasso.txt do passo a passo do processo.
