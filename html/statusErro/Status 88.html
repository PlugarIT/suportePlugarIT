Status 88

O aplicativo encontrou um erro de modo incompatível

    Em geral esse código retorna quando o arquivo esta sendo usado por um usuário de modo exclusivo.

    Isso pode acontecer por exemplo se um usuário fecha o sistema pelo X da janela, dessa forma o arquivo utilizado não é encerrado naquele momento, ficando preso ao uso exclusivo do usuário.

    DICA: Pedir para todos os usuários fecharem todos os sistemas e entrarem novamente, caso seja possível verificar no Gerenciamento do Servidor se tem arquivos abertos e finalizar.
