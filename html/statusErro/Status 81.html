Status 81

O Microkernel encontro um erro de bloqueio - Lock error - este erro acontece no btrieve 6.15

* Status 81 e erro conectcont.dll no login

    SOLUÇÃO:

  • Para testar renomeie os arquivos CNCVINCE.BTR e CNCUSUAR.BTR, apague também os arquivos com extensão LCK.
  • Caso não dê certo recorte todos os arquivos que começam com CNC para uma outra pasta e teste novamente.
  • Caso não resolva renomeie a pasta Geral e crie uma nova, copie o arquivo VERINFO.BTR para a nova pasta e teste novamente.
  • Se resolver copie os arquivos que começam com a letra A e teste, caso não dê erro copie os que começam com a letra B, assim em diante até descobrir qual arquivo está causando o erro.

* Status 81 - Lock Error

    DICA: A princípio apagar os arquivos LCKs, pode ser que excedeu a quantidade setada no registro, caso não resolva reiniciar as máquinas, pode ser que caiu a conexão de algum usuário e o arquivo ficou preso.
