  • A Empresa Btrieve foi adquirida pela Novell em 1987, mas desmembrou-se em 1994 como uma nova empresa chamada Btrieve Technogies (liderada por Doug e Nancy Woodward e Ron Harris). Após algum tempo a Btrieve Technologies foi renomeada para Pervasive Software.

  • O Btrieve é ​​um banco de dados transacional baseado no método de acesso sequencial indexado , que é uma forma de armazenar dados para recuperação rápida. Houve várias versões para DOS , Linux , também para Microsoft Windows, Windows 98, Windows NT, Windows 2000 , Windows XP , Windows Server 2003 e para Novell NetWare

  • O Btrieve foi originalmente um recordista que foi publicado pela SoftCraft na mesma época do lançamento dos primeiros PCs IBM.

  • Depois de ganhar participação de mercado e popularidade, foi comprado pela Novell para integração do sistema operacional Netware. O produto não conseguiu ganhar participação de mercado significativa e, após alguma reorganização dentro da Novell, o produto foi feito para ser desenvolvido por uma nova empresa conhecida como Btrieve Technologies, Inc. (BTI).

  • O Btrieve foi modularizado a partir da versão 6.15 e se tornou um dos dois bancos de dados centralizados em uma interface de software chamada Micro-Kernel Database Engine. O outro produto foi o Scalable SQL , um banco de dados relacional que utilizava a Linguagem de Consulta Estruturada . .

  • Depois que várias novas versões foram lançadas, a empresa foi renomeada para Pervasive Software e agora é vendida pela Pervasive PSQL

  • Software Pervasive é o desenvolvedor do Integrador de Dados Pervasive ou anteriormente chamado Business Integrator. Agora, é adquirida pela Actian Corporation.
