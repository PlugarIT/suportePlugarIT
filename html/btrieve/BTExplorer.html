BTExplorer

O BTExplorer é uma ferramenta para abrirmos os arquivos e verificar se há sujeira gravada

Como utilizar o BTExplorer

  • Copie o executavel do Bt para uma pasta chamada BTExplorer

  • Para utilizar o BTExplorer é necessário que tenha no mesmo local do arquivo a ser aberto oa DDFs correspondente.

  • Execute o BTExplorer e abra o menu Arquivo Configurar Diretórios:

  • Faça o apontamento para a pasta que se encontra as DDFs relacionadas ao arquivo:

  • Após isso basta clicar 2x sobre o arquivo para abrir
