06 - Como reiniciar o serviço de certificado

SOLUÇÃO:

  • Identificando o PID para matar o processo ( CertPropSvc)

  • Abrir o gerenciado de tarefas na guia serviços, encontrar o processo informado acima e identificar o pid do serviço.

  • Identificado o processo e o PID, abra o CMD como ADM,

  • Digitar o seguinte comando [ TASKKILL /F /PID [Codigo PID do serviço] no CMD] e de enter para matar o processo,

  • após matar o processo, copiar o arquivo [svc-status] pra aréa de trabalho do servidor que apresentou problema, para adcionar certigficado no perfil do cliente e executrar como ADM.

  • arquivo se encontra [\\cmatic-ad-01\c$\CMATIC - Scripts\Reiniciar Servico Certificado].
