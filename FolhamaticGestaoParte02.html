Antes de começarmos a segunda etapa temos que fazer alguns ajustes no diretório do Phoenix conforme orientações abaixo:

  • Entre na pasta Phoenix, depois na pasta Geral e procure pelo arquivo Config.ini

  • Modifique o campo Habilitada para 1 isso vai habilitar na Folha o campo de importação

  • Agora entre na pasta \\contspfil01\Suporte_Redes\CONVERSAO\Conversao Sage SQL Server\DLL_MigracaoSage e copie as 2 DLLs para dentro da pasta Phoenix

  • Agora entramos na pasta Phoenix e executamos o Exportacao.exe

Nesta tela podemos testar a conexão com o SQL

Agora iremos configurar a tela para gerarmos os Jsons

  • Origem da Exportação: Sempre será Sage

  • Produto: Gestão Contabil Sage

  • Tipo de Autenticação: SQL Server

  • Servidor: .\SQLEXPRESS

  • Banco de Dados: Sage_Gestão_Contabil

  • Usuário: sa

  • Senha: sa

  • Path JSON: Já vem configurado>

Teste a conexão com o Banco de Dados

Clique em Avançar

Irá mostrar os dados que serão convertidos do Gestão

Clique em Exportar

No final do processo será listado os cadastros concluídos com sucesso.
