Erros Backup

CLIQUE AQUI - Como fazer o agendamento do Backup

01 - Erro Out Of Memory

- Este erro acontece quando a base do cliente é muito grande, devido o sistema ser em 32bits, a programação esta trabalhando nisso.

02 - Não esta fazendo o Backup Agendado

- Verifique no Agendador de terefas do Windows se existe a terefa agendada.

- Entre no agendador de Tarefas do Windows e verifique se a tarefa esta lá.

- Detalhes do agendamento.

03 - Btrieve Status 20 no Btrieve 12

    SOLUÇÃO:

  • Verificar se o btrieve 12 está instalado.

  • Verificar se tem as licenças instaladas em License Administrator.

  • Verificar se o serviço do Btrieve 12 esta iniciado.

04 - Btrieve Status 20 no Btrieve 6.15

    SOLUÇÃO:

  • Desinstale e reinstale o Backup.
