Status 45

Os flags de chave especificados são inválidos

The specified key flags are invalid

    A especificação dos flags de chave em uma operação Create é inconsistente. Se uma chave tem vários segmentos, os atributos duplicate, modifiable e null precisam ser os mesmos para cada segmento na chave. Também não se pode usar os atributos de chave null ou manual em um arquivo somente de chaves. O MicroKernel também retorna este código de status se o aplicativo tentar especificar seqüências de ordenação alternativa (Alternate Collating Sequences, ACS) diferentes para dois ou mais segmentos de uma chave segmentada.
