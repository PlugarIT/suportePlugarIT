Run Time Error

01 - Run Time Error 217 Btrieve 6.15

    DICA: Refazer o registro do Midas.dll

    Veja na página inicial item Midas Dll

02 - Run Time Error 217 Btrieve 6.15 - Abre o sistema apenas em um computador

    DICA: Refazer as permissões e reiniciar o servidor

03 - Run Time Error no Btrieve 12.

    DICA: Este caso pode ser vários fatores, por isso verificar os itens abaixo.

  • Colocar em Propriedades - Iniciar em: o local do exe.

  • Colocar em Propriedades - O endereço IP do servidor no lugar do nome.

  • Verificar se tem arquivos danificados na pasta Geral.

  • Verificar permissões na pasta Phoenix.

  • Verificar se o serviço B12 esta iniciado.

  • Verificar se o B12 está instalado na estação.

  • Verificar se o Firewall, criar regra de entrada e saída para o B12.

  • Verificar se o Arquivo PVSW~.Loc na pasta Geral esta com o nome do servidor.

  • Verificar as Permissões no registro.

  • Baixar o AC do usuário.

  • Verificar se existem as Dlls e exe do btrieve 6.15 w32mkde.exe w34mkrc.dll.

Caso não resolva, tente fazer o seguinte.

    Entre em Btrieve Control Center.

    Clique com o botão direito do mouse em Engine.

    Clique em New Server.

    Digite o nome correto do serviidor.

    Provavelmente irá pedir o login e senha de acesso do servidor.

    Tente entrar no sistema novamente.

04 - Run Time Error 216

    DICA: Verificar se tem impressora definida como padrão e também data e hora do computador.
